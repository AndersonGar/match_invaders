# Invader Match

This is a videogame example about Space Invaders with mechanics from Match3 like Candy Crush.


![image](https://user-images.githubusercontent.com/48804461/181641842-78e86447-6875-4b79-b917-0bde484cba9a.png)


## Controls

- Move Rigth: D or right arrow
- Move Left: A or left arrow
- Shoot: Space key
